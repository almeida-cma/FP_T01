História e Evolução das Linguagens

Linguagem de programação é um conjunto de instruções que dizem ao computador o que fazer. Nas décadas de 1940, usava-se linguagem de máquina (binário). Surgiu então o Assembly, com mnemônicos.

1940s Linguagem de máquina (binário)
1950s Assembly, FORTRAN, COBOL, ALGOL
1970s C, Simula 67 (classes), Smalltalk (POO)
1980-90s C++, Python, Java, JavaScript, PHP
2000+ C#, Ruby, Swift, Kotlin, TypeScript

Vídeos: O que é Programação? | Principais Linguagens

Níveis de Programação

🔻 Baixo nível: Próximo ao hardware (Assembly).
🔺 Alto nível: Próximo à linguagem humana (Python, Java, JavaScript, C#).

Compilador x Interpretador:
Compilador: traduz todo o código antes da execução (C, C++).
Interpretador: traduz e executa linha a linha (Python, JavaScript).

Paradigmas de Programação

  • Estruturada: sequência, decisão, repetição
  • Modular: código dividido em módulos
  • Orientação a Objetos: objetos com atributos/métodos
  • Orientação a Eventos: fluxo guiado por ações do usuário

Lógica e Algoritmo

Lógica: conjunto ordenado de instruções para alcançar um objetivo.
Algoritmo: sequência finita de passos para resolver um problema.
Código-fonte: texto escrito pelo programador.
Programa executável: código convertido para linguagem de máquina.

Portugol Webstudio - Sintaxe Oficial

O Portugol Webstudio (portugol.dev) é um ambiente online moderno. Sintaxe correta com indentação:

programa { funcao inicio() { // seu código aqui escreva("Olá, mundo!") } }

Exemplo 1: Saudação

programa { funcao inicio() { escreva("Olá, mundo da programação!") } }

Exemplo 2: Entrada de Dados

programa { funcao inicio() { cadeia nome escreva("Digite seu nome: ") leia(nome) escreva("Olá, ", nome, "! Bem-vindo ao Portugol!") } }

Exemplo 3: Estrutura Condicional (se/senao)

programa { funcao inicio() { real nota escreva("Digite a nota do aluno: ") leia(nota) se (nota >= 7) { escreva("Aprovado! Parabéns!") } senao { escreva("Reprovado. Estude mais!") } } }

Exemplo 4: Laço de Repetição (para)

programa { funcao inicio() { para (inteiro i = 1; i <= 5; i++) { escreva("Contagem: ", i, "\n") } escreva("Fim da contagem!") } }

Exercícios Resolvidos (Portugol Webstudio)

1. Ditado Popular

programa { funcao inicio() { escreva("A persistência é o caminho do êxito.\n") escreva("Quem espera sempre alcança.\n") escreva("A pressa é inimiga da perfeição.") } }

2. Par ou Ímpar

programa { funcao inicio() { inteiro num escreva("Digite um número: ") leia(num) se (num % 2 == 0) { escreva(num, " é PAR") } senao { escreva(num, " é ÍMPAR") } } }

3. Média com Aprovação

programa { funcao inicio() { real n1, n2, n3, media escreva("Nota 1: ") leia(n1) escreva("Nota 2: ") leia(n2) escreva("Nota 3: ") leia(n3) media = (n1 + n2 + n3) / 3 se (media >= 7) { escreva("Aprovado! Média: ", media) } senao { escreva("Reprovado. Média: ", media) } } }

4. Tabuada

programa { funcao inicio() { inteiro num escreva("Digite um número: ") leia(num) para (inteiro i = 1; i <= 10; i++) { escreva(num, " x ", i, " = ", num * i, "\n") } } }
